
Welcome to **BreezeUI**! This library is designed to help developers build sleek, accessible UIs for data-rich applications — fast.

### 🚀 Features
- 20+ ready-to-use, customizable components
- Tailwind CSS + React + TypeScript support
- Dark mode built-in
- WCAG-compliant accessibility
- Lightweight and fast

### 📦 Installation

```bash
npm install @breezeui/react
# or
pnpm add @breezeui/react
```
